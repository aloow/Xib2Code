Xib2Code
==========
Xib2Code可以把项目中Xib文件转换成代码形式
* 支持选择，选择要转换的xib文件，转换成OC的.h和.m文件。
* 支持直接输入xml格式的文本，转换成OC的.h和.m文件。

### 使用
在一个文件下，命令行输入 git clone https://github.com/aloow/Xib2Code.git
然后打开项目中Xib2Code.xcworkspace文件，最后Command + Run，跑项目。
